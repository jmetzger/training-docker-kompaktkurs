# Ubuntu HelloWorld 

## Simple Version 

### Schritt 1:
```
cd 
mkdir Hello-World 
cd Hello-World
```

```
## Schritt 2:
# nano Dockerfile
FROM ubuntu:latest 

COPY hello.sh .
RUN chmod u+x hello.sh
CMD ["/hello.sh"]
```

### Schritt 2:
```
nano hello.sh 
```

```
#!/bin/bash
let i=0

while true
do
  let i=i+1
  echo $i:hello-docker
  sleep 5
done
```
